# Project Brief: CCR Tools

## What we are building
A bulk upload tool for Clinical Case Note Review (CCR) survey data. Clinicians collect one record per patient and wish to submit data in bulk via an Excel template, rather than entering records individually through TBN's online data collection methodology.

## Why we are building it
TBN's existing online methodology does not support bulk upload. To meet clinician needs, Excel templates have been distributed as a workaround. Alex has built a working Excel/VBA tool for one project. That build is tightly coupled to its specific template and cannot be reused as-is. Two further projects now need equivalent tools.

## What success looks like
- A working bulk upload tool for each of the two new projects
- Clinicians can complete the Excel template and upload data without errors
- The tools are tested and verified before use with real data

## Constraints and context
- Platform is Excel/VBA, following Alex's existing approach
- Alex is on paternity leave — his build is a reference
- The detail of Alex's code is not yet fully understood — an exploratory phase is needed before development begins
